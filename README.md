# Netflix Content Analysis 🎬

## Overview
Exploratory Data Analysis (EDA) on Netflix titles dataset to uncover content trends, viewer patterns, and platform growth insights using Python.

## Tools & Libraries
- Python, Pandas, NumPy
- Matplotlib, Seaborn, Plotly

## Dataset
- Source: netflix_titles.csv (Kaggle)
- Contains: Title, Type, Country, Release Year, Rating, Director, Date Added

## Analysis Performed
- Movies vs TV Shows distribution
- Top 10 countries producing Netflix content
- Content growth trend by release year
- Content added to Netflix over the years
- Rating distribution across content
- Top directors on Netflix

## Key Insights
- Movies dominate Netflix content compared to TV Shows
- USA leads in content production followed by India
- Netflix content additions peaked significantly post-2015
- TV-MA and TV-14 are the most common ratings

## How to Run
1. Clone this repository
2. Install dependencies: `pip install pandas matplotlib seaborn plotly`
3. Run: `python netflix_analysis.py`
